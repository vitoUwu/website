## How to run it locally

### Dependencies

[Bun](https://bun.sh/)

[Git](https://git-scm.com/)

### Clone

```bash
git clone https://github.com/vitoUWu/website && cd website
```

### Running

```bash
bun run dev
```
Now enter `localhost:3000` to access locally
